# 📊 Customer Retention & Churn Analysis

## 🔍 Project Overview

This project analyzes customer behavior using e-commerce data to understand retention, churn, and customer value.

## 🎯 Objective

* Identify repeat vs one-time customers
* Measure retention and churn rate
* Analyze customer trends over time
* Discover high-value customers and regions

## 🛠️ Tools Used

* Power BI
* Excel / CSV

## 📊 Key Metrics

* Total Customers
* Repeat Customers
* Retention Rate
* Churn Rate

## 📈 Insights

* Majority of customers are repeat buyers
* Repeat customers contribute most revenue
* Customer activity shows an increasing trend
* Certain countries dominate revenue

## 🚀 Key Takeaways

* Focus on converting one-time customers
* Improve retention strategies
* Target high-value customers
* Optimize regional marketing
